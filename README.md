# ChecklistBug

Guia detallada para auditorias de seguridad web y preparacion para la certificacion CBBH. El proyecto centraliza tecnicas de pentesting, gestion de vulnerabilidades y puntos de verificacion criticos en aplicaciones web.

## Contenido
- Checklist detallada de auditoria web
- Referencias de practica en VulNyx
- Metodologias de footprinting y explotacion
